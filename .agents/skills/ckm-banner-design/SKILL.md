# CKM Banner Design

Guides banner creation across social media, web, display ads, and print using
platform-specific size specs and 22 art direction styles.

---

## Design Process

### 1. Confirm Platform & Purpose

If not specified, ask:

- Platform (Facebook, LinkedIn, YouTube, Instagram, web, print, etc.)
- Purpose (brand awareness, event, product launch, CTA, etc.)
- Destination format (static image, animated, print-ready)

### 2. Select Size

Read `references/banner-sizes-and-styles.md` for the canonical size list.
Common starting points:

- Social cover → platform-specific (see Social Media table)
- Display ad → GDN sizes (Medium Rectangle 300×250 is the default)
- Web hero → full-width 1920×600–1080 or section 1200×400
- Print → confirm format and apply 300 DPI / 3–5mm bleed

### 3. Choose Art Direction Style

Present 2–3 options from the 22 styles in the reference, ranked by fit for the
stated purpose. Require explicit selection before proceeding.

### 4. Produce Design Brief

```text
Platform:      [platform]
Size:          [W × H px or mm]
Style:         [style name]
Headline:      [≤ 7 words]
Sub-copy:      [≤ 3 lines]
CTA:           [action verb + outcome]
Color palette: [3 values max]
Typography:    [max 2 typefaces]
Safe zone:     [central 70–80% of canvas]
```

---

## Constraints

- Never fabricate or expose personal data
- One CTA per banner
- Text under 20% of canvas area for ad formats
- Minimum 4.5:1 contrast ratio for all text
- Critical content in central 70–80% of canvas (safe zone rule)
- Max 2 typefaces per banner
