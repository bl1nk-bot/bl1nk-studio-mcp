# thai-answer.md

กฎสำหรับการตอบคำถามเป็นภาษาไทย

## Brief overview

- กฎนี้กำหนดให้ Kilo Code ตอบคำถามเป็นภาษาไทยเสมอเมื่อทำได้

## ภาษาที่ใช้

- **ภาษาหลัก**: ภาษาไทย - ใช้ตอบคำถามและอธิบายทุกอย่างเป็นภาษาไทย
- **Code/Technical Terms**: สามารถใช้คำศัพท์ภาษาอังกฤษที่เป็นที่ยอมรับในวงการเทคนิคได้ เช่น API, JSON, TypeScript, React เป็นต้น

## ตัวอย่างการใช้

- แทนที่จะตอบ "I've created the file" ให้ตอบ "ผมสร้างไฟล์สำเร็จแล้ว"
- แทนที่จะตอบ "This is a bug" ให้ตอบ "นี่คือบั๊ก"
- สามารถใช้รวมกันได้ เช่น "ไฟล์ TypeScript ถูกสร้างแล้ว"

## Trigger Cases

- เมื่อผู้ใช้ถามคำถามใดๆ ให้ตอบเป็นภาษาไทยเสมอ
- เมื่ออธิบายการทำงานของ code ให้ใช้ภาษาไทย
- เมื่อแสดงผลลัพธ์หรือสรุปงาน ให้ใช้ภาษาไทย
